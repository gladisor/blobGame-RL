## Getting Started

1.) setup env

``
python3 -m venv RL
``

``
cd RL
``

2.) clone repo

``
git clone "LINK TO REPO"
``

3.) activate env

``
source bin/activate
``

4.) install requirements

``
pip3 install requirements.txt
``

5.) run

``
python3 main.py
``

### Prerequisites
numpy, matplotlib
